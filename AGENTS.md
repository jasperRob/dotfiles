# Agent Guidelines

## Documentation

When making any changes to this repo, update `README.md` to reflect those changes:

- **New config added** — add or update the relevant section under `## Configuration Details` with a description and notable settings. Add the symlink to the table in `## Installation` if `install.sh` installs it.
- **Config modified** — update its existing section if behaviour, keybindings, or notable settings changed.
- **Config removed** — remove its section from the README and its symlink row from the installation table.
- **`install.sh` changed** — update the symlink table in `## Installation` if any symlinks were added, removed, or changed.
- **New dependency** — add it to the appropriate list under `## Dependencies`.

Keep README sections consistent with the existing format: a heading with the tool name, a short description, and a table or bullet list of notable bindings/settings where applicable.
